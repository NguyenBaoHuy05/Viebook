# Viebook

## Ứng dụng web dành cho người Việt chạy bằng các công nghệ như Laravel(PHP), Nextjs, Reverb serve

+ Cách chạy frontend
```
npm install
npm run dev
```
+ cách chạy backend
```
composer install
php artisan serve
php artisan reverb:start --debug
```
+ Cách chạy AI
```
pip install -r requirement.txt
```
+ Khởi tạo serve của FastAPI
